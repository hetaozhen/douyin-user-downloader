# Douyin User Downloader  
**抖音用户主页批量无水印下载器**

基于强大的 Playwright 和 Python 打造的自动化数据采集工具。它可以全自动、无感地批量抓取并下载指定抖音用户主页的**所有视频**和**所有图文作品**。

---

## 🌟 核心特性 (Key Features)

- **突破 CDN 视频防盗链限制**：完美伪装成真实的 Chrome 浏览器发包，附带底层核心认证请求头 (`User-Agent` 与 `Referer`)，彻底解决纯视频被拦截报错 `403 Forbidden` 的难题。
- **智能虚拟列表 (Virtual List) 滑动机制**：舍弃了传统的、容易误判的 DOM 高度检测。直接向黑盒容器中输入物理 `鼠标悬浮滚轮` 及 `定向 JS 下拉`，强制触发抖音最高强度的动态加载，确保 100% 渲染出所有新视频。
- **神级防漏底层网络断流检测**：基于最底层的 `/aweme/v1/web/aweme/post/` API 数据包进行监听，它完全不在意前端网页看起来有多卡顿，只要它连续 15 秒没通过暗网收到任何含新作品的数据流才会聪明地说再见，不落下一个视频。
- **历史登录保护（静默后台版）**：支持持久化存储 Cookie，只需首登一次，往后默认转入隐形黑客模式 (`Headless Mode`) 后台无闪烁光速拉取数据；遇到罕见的超级防爬滑块验证码，也可随手切出窗口用鼠标一划即解。
- **极限并发与自动断点重试**：内建最高效的线程池池并发处理模块，遭遇丢包或限流时能智能自动等侯 2 秒二次发起下载，无惧极端恶劣的网络状况。

## 🚀 快速启动指南 (Quick Start)

### 1. 安装环境依赖
你需要 Python 和内置的 Playwright 驱动环境。
```bash
pip install playwright requests
playwright install chromium
```

### 2. 开始抓取！
在终端中启动主程序，并在提示处输入你需要下载的主页链接即可：
```bash
python douyin_user_downloader.py
```
> 或者直接附带链接运行：
> `python douyin_user_downloader.py "https://www.douyin.com/user/xxxxxx"`

### 3. 数据去哪里了？
抓取下来极度干净的高清图片、无水印视频将会根据作者的名字自动分类归档存放在你脚本同目录下的 `downloads/作者名/` 专用文件夹中！
